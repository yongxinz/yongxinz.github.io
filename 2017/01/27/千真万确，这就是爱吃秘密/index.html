千真万确，这就是你爱吃的秘密

发表于 2017-01-27 | 分类于 读书 | | 阅读次数

过年期间，除了吃和打麻将，好像也没有什么其他事情要做。大家见面除了彼此问候过年好之外，最常说的可能就是：「我可能又变胖了」。面对这么残酷的现实，喊出：「我要减肥」或者「我不能再吃了」的口号是必须的，剩下的就是那张管不住的嘴了。

每个人都爱吃，也喜欢吃，但这都不是你的错，这是祖先给我们留下的一项技能，关于生存的技能，具体来说是一种叫做「贪吃基因」的东西在作怪。

这个观点来自于《人类简史》，2016 年非常火的一本书，作者是尤瓦尔·赫拉利，一名以色列作家，非常的博学，对历史学、人类学、社会学、宗教等内容信手拈来。更令我惊讶的是，作者会根据出版国家的不同，进行相应内容的定制，由此可见，知识面之丰富可谓海量。

作者把人类发展分成三个阶段：认知革命，农业革命和科学革命。在认知革命章节，作者讲了这样一个故事：我们为什么如此爱好甜食，因为甜食代表着高热量，在我们的采集者祖先时代是永远都供不应求的维生物品。如果一个石器时代的女性碰到了一棵长满甜美无花果的树，她最明智的做法就是立刻吃到吃不下为止，否则等到附近的沸沸也发现了这棵树，她可能一颗也吃不到了。

这种大口吃下高热量食物的直觉本能深植在我们的基因里，是经过数百万年演化的结果。食物丰富的现在也同样如此，我们喜欢吃大鱼大肉，很少能够吃一口就停下来，往往是已经腻到再也吃不下了才决定不吃。一切可能都源于我们的基因中依旧记得曾经那些食不果腹的日子。

但是，吃是人生的一大乐事，特别是过年，怎么能不吃呢？反正我怎么吃都不胖，你随意。
